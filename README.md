# Mumboline

## This project uses TWILIO API's to allow users to buy and use temporary numbers for their businesses

### Technologies used

Frontend :

- React
- Material UI
- Neomorphic UI

Backend :

- Node.Js
- Express.Js
- Passport
- Axios
- Bcrypt
- Semaphores

Services :

- Twilio API
- Stripe API

The documentation for the Twilio API can be found [here](https://www.twilio.com/docs/all).

Database :

- MongoDB

Email Handling :

- Twilio's Sendgrid API

Deployment :

- Digital Ocean Droplets

## Installation

`npm run dev`

## Node Version

`node >= 6.0.0`

## Run

`npm install`
